## 概要

趣味で開発するジュエリーECサイト。

公開は予定していない。

目的は、Next.jsとLaravelを用いたフルスタック開発、およびECサイトの設計・実装を学ぶこと。

---

## 特徴

- 在庫管理
- 商品管理
- ジュエリーの3D表示（Blender）
- 鑑定書PDFの自動生成
- PDFダウンロード
- メール送付
- オンライン相談予約
- 管理画面(COMなど)
