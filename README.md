# Clarijewel

A project for practicing IT infrastructure and web application development.

## Tech Stack
- **Backend**: Laravel (PHP)
- **Frontend**: Next.js (TypeScript)

## Getting Started

### Backend
1. `cd backend`
2. `php artisan serve`

### Frontend
1. `cd frontend`
2. `npm run dev`
